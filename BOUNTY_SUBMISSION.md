## 🛡️ GitVuln Watch — AI-Powered Open Source Vulnerability Intelligence 🌰

**Live Dashboard**: https://lalalic.github.io/gitvuln-watch/
**Repository**: https://github.com/lalalic/gitvuln-watch

### 🌰 What It Does

GitVuln Watch monitors the **GitHub Advisory Database** for critical and high-severity vulnerabilities across **6 package ecosystems** (npm, pip, go, cargo, nuget, rubygems) and generates AI-powered weekly intelligence briefs via **GitHub Models (GPT-4o-mini)**.

### 🌰 Why This Is Different

While every other submission monitors crypto/DeFi/blockchain, GitVuln Watch addresses a completely different problem: **open source supply chain security**.

| Aspect | Other Submissions | GitVuln Watch 🌰 |
|--------|-------------------|-------------------|
| **Domain** | Crypto exchanges, DeFi, wash trading | Open source package security |
| **Data Source** | CPW Tracker API | GitHub Advisory Database (free!) |
| **Target Users** | Traders, compliance teams | Software developers, DevSecOps |
| **Threat Type** | Market manipulation, hacks | CVEs, supply chain attacks |
| **Ecosystems** | Crypto exchanges | npm, pip, go, cargo, nuget, rubygems |

### 🌰 Key Features

- **6 Ecosystem Coverage** 🌰 — npm, pip, go, cargo, nuget, rubygems
- **Real CVSS Scoring** 🌰 — Actual vulnerability severity scores from GitHub
- **AI Intelligence Brief** 🌰 — Weekly risk assessment, key findings, trends, recommendations
- **Ecosystem Risk Matrix** 🌰 — Per-ecosystem risk evaluation from AI
- **Social Media Thread** 🌰 — Auto-generated Twitter/X security content
- **500-Advisory Rolling Archive** 🌰 — Historical trend tracking
- **Interactive Dashboard** 🌰 — Filter by ecosystem, CVSS scores, GHSA links
- **Zero-Cost Infrastructure** 🌰 — GitHub Advisory Database is free!

- **Automation**: GitHub Actions weekly cron + GitHub Pages deploy
- **Zero npm dependencies** — Pure Node.js ESM
